Экспорт данных

Скачайте данные для анализа в Excel, Python или других инструментах. Этот блок экспортирует прогноз (Forecast), а не историю наблюдений (History).

Параметры экспорта

Данные включают: временные метки, координаты, AQI, все загрязнители (PM2.5, PM10, NO2, SO2, O3) и НМУ риски.

Форматы файлов

Статус экспорта: ожидание действия.

Privacy-first: координаты включены только для анализа, личные данные не собираются. Доступный горизонт может быть меньше запрошенного.

Forecast: почасовой прогноз качества воздуха

Просмотрите ожидаемую динамику качества воздуха на ближайшие часы.

Выберите горизонт для просмотра прогноза
